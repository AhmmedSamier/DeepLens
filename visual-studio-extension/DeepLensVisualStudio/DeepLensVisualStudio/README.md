# DeepLens for Visual Studio

DeepLens is a powerful "Search Everywhere" extension for Visual Studio, inspired by modern IDE navigation features. It provides instant access to files, classes, symbols, and API endpoints across your entire solution.

## Features

- **Double Shift Activation**: Press `Shift` twice quickly to open the search window instantly.
- **Unified Search**: Search for everything in one place - files, types, members, and more.
- **CamelHumps Support**: Type `UB` to find `UserButton` (camelCase abbreviation matching).
- **Modern UI**: A clean, responsive interface that matches Visual Studio's theme (Dark/Light).
- **Filter Scopes**: Quickly filter results by type (Classes, Files, Symbols, Endpoints, etc.).
- **Preview & Navigation**:
  - Single-click to open files at the specific line.
  - Hover actions to "Reveal in File Explorer" or "Copy Path".

## Keyboard Shortcuts

- **Shift + Shift** (Double Press): Open Search Window (Global)
- **Arrow Keys**: Navigate results
- **Enter**: Open selected result

## Installation

1. Install from the Visual Studio Marketplace (coming soon).
2. Or download the latest `.vsix` from GitHub Releases.
3. Double-click the `.vsix` file to install.

## Requirements

- Visual Studio 2022 (v17.0+)
- .NET Framework 4.7.2+
- **Bun** (Preferred) or **Node.js**: Required to power the high-performance search engine.
